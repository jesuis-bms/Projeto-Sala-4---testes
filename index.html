Quem nós somos?

O Projeto Sala 4 é um projeto desenvolvido pelos estudantes Bernard Marques, Lucas Reis, Esther Gomes e Jorge para, inicialmente, ajudar os alunos da Escola Municipal Dom Bosco no que diz respeito à acessibilidade.

Onde podemos saber mais?

O projeto é um open-source e seu código está disponpivel no nosso GitHub !
